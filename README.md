Spatial AI Canvas
A full-stack intelligent workspace for real-time visual reasoning. Users upload course or project files, ask questions by text or voice, and receive AI-generated explanations with diagrams drawn directly on an interactive canvas.
What it does

Upload documents and ask questions about them via text or voice
AI responds with structured explanations rendered as diagrams on the canvas
JWT-based auth with grouped project management
Multi-model LLM support (OpenAI / Ollama)
Document text extraction for context-aware responses

Tech Stack
Frontend: React, TypeScript, Vite, tldraw (infinite canvas)
Backend: FastAPI, SQLite
AI: OpenAI API, Ollama, LangChain
Auth: JWT
